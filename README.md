# Виджет банковских операций
## Проект представляет собой интерфейс для отображения банковской активности клиентов. Поддерживается фильтрация и сортировка транзакций по статусу и дате соответственно.

### Установка и запуск
Перед использованием убедитесь, что выбранная версия Python >= 3.8.

* Склонируйте репозитории:

git-клон https://github.com/Vissashka/domashka.git

* Устанавливаем виртуальное окружение и зависимости

cd domashka 
poetry install

pip install -r requirements.txt

* Запустите приложение (пример запуска скрипта):
python app.py

### Функционал сервиса

Сервис включает следующие возможности:

* Маскировка номеров карт и счетов: используется для защиты пользователей цифровых данных.
* Фильтрация транзакций: позволяет выбирать только успешные операции.
* Сортировка транзакций: сортирует операции по дате (самые свежие продукты вверху).
### Использование
* APIМодуль src/masks.py обрабатывает персональные данные (карты и счета)

* модуль src/widget.py формирует основной виджет

* модуль src/processing.py ступень обработки транзакций.

## Дополнительные модули
 Помимо базовых модулей, добавился дополнительный важный компонент — модуль src.generators, который обеспечивает эффективные средства для работы с большими объёмами данных транзакций и поддерживает обработку больших наборов данных через механизмы генераторов.

### Модуль src.generators
Описание: Модуль реализует мощные инструменты для гибкого управления и обработки данных транзакций, включая фильтры и генерацию номеров банковских карт.
### Основные компоненты:
* filter_by_currency(): Функция, принимающая список транзакций и код валюты, возвращает итератор, выдающий только те транзакции, чьё поле валюты соответствует заданному значению.
* transaction_descriptions(): Функция-генератор, принимающая список транзакций и последовательно выдающая их описания путём использования оператора yield.
* card_number_generator(): Генератор, способный создавать номера банковских карт в нужном диапазоне, передаваемом через параметры start и stop.
